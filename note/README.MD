arduino的基本概念就是:
  可以由程式控制每個引腳的電壓變化的工具
  語言是c++
  但是每個引腳各自有不同的能力,只有有~圖示的才有高頻開關的能力

platformio的基本概念就是:
  他是一個在vscode裡面的可以自動根據硬體變化的綜合型工作台
  可以做編輯c++(有自動補全),
  自動搭建好對應的環境,並且可以抓取對應的c++函式庫
  將c++編譯成arduino可以懂得機器碼
  燒錄進arduino和監視操控

  要寫程式要去src>main.cpp寫
  要創建新的專案要用他自帶的new project
  要編譯:左邊圖示>general>Build
  燒錄:左邊圖示>general>Upload
  監視操控:左邊圖示>general>Monitor

紅外線接發基本概念:
  有規範統一用38khz的紅外線(干擾最少),且有規定什麼英文字母對應到的閃爍摩斯密碼和對應32位元數字(每次讀取32位元)
  我們寫程式時仍用發射或接收0xXX(用0x開頭後面16進制),這樣arduino比較好編譯,不會翻譯錯誤
  記得要記住上一次的開關狀態,不然你不小心有一直碰著他就一直觸發
  開關他有INPUT_PULLUP和INPUT可選:
    INPUT_PULLUP:沒用時是高電壓,按下去低電壓,預設用這就好
    INPUT:沒用時是低電壓,按下是高電壓,很敏感,不要用
  紅外線發射函式IrSender.sendNEC(address, command, repeats)：
    address (0x00)：這是這台遙控器的「身分證」(要先設定)
    command (0x01~0x04)：這是具體的「命令內容」
    repeats (0)：代表不重複發送。如果你長按按鈕想要連續加音量，以後可以調整這裡
  紅外線接收函式IrReceiver.decode()
